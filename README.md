## ğŸš¦ Getting Started

### Prerequisites
- Node.js >= 20.0.0
- pnpm >= 9.0.0

### Installation & Setup
```bash
# Clone the repository
git clone https://github.com/saswatawork/codecraft-labs.git
cd codecraft-labs

# Install pnpm if you haven't already
npm install -g pnpm@latest

# Install dependencies
pnpm install

# Setup git hooks
pnpm prepare

# Start development servers
pnpm dev

# Build all packages
pnpm build

# Run tests
pnpm test

# Format and lint code
pnpm check
```

### Development Workflow
```bash
# Start specific app
pnpm --filter portfolio dev
pnpm --filter web dev

# Build specific package
pnpm --filter @ccl/ui build

# Add dependencies
pnpm --filter portfolio add sharp
pnpm --filter @ccl/ui add -D @types/node

# Run tests in watch mode
pnpm test:watch

# Storybook
pnpm --filter @ccl/ui storybook
```
